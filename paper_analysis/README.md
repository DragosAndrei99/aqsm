# Paper Analysis Assets

This directory contains the paper, extracted source artifacts, and local analysis notes for the AQSM/HDWM implementation.

## Contents

- `adaptive quantum scaling model.pdf`: local copy of the paper.
- `paper_analysis.md`: summary of the paper, implementation interpretation, and reproducibility gaps.
- `aqsm_2x2_step_by_step.md`: classical `2x2 -> 4x4` worked example.
- `aqsm_2x2_quantum_step_by_step.md`: quantum-notation companion walkthrough.
- `arxiv_src/`: extracted arXiv source bundle.
- `adaptive quantum scaling model.txt`: extracted PDF text.
- `adaptive_quantum_scaling_model_arxiv_source.tgz`: downloaded arXiv source archive.
- `aqsm_fig*`: extracted figure helper files.

The runnable project entrypoints remain at the repository root:

- `README.md`
- `main.py`
- `aqsm_watermark_embedding_walkthrough.ipynb`
